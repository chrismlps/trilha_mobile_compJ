## Zeus - Sistema de Gestão Interna (Comp Júnior)
Uma aplicação mobile desenvolvida para o Desafio Final da trilha de Mobile da Comp Júnior. O sistema foca no gerenciamento e organização de informações internas da empresa, com seis categorias: Membros, Penalidades, Eventos, Projetos, Clientes e Orçamentos.

## Tecnologias Utilizadas
React Native com Expo: Utilizado para agilizar o desenvolvimento e permitir a visualização em tempo real através do Expo Go.
React Navigation: Biblioteca escolhida para gerenciar a transição entre as seis áreas do sistema via navegação por abas.
JavaScript: Linguagem base para a construção da lógica do front-end.

## Justificativas Técnicas
Escolha do Expo: A utilização do Expo foi definida pela facilidade de teste em múltiplos dispositivos e pela agilidade no ciclo de desenvolvimento (Fast Refresh), permitindo focar exclusivamente no front-end mobile como solicitado pelo desafio.
Arquitetura de Componentes: O projeto foi estruturado de forma componentizada para garantir o reaproveitamento de código, facilitando a manutenção e garantindo uma identidade visual consistente em todas as telas.
Uso de Mocks: Devido ao escopo do projeto não exigir back-end, os dados são simulados localmente através de objetos JSON, garantindo a performance e a autonomia da aplicação
